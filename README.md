# LLM RAG Evaluation Framework

This repository contains end-to-end test automation for RAG (Retrieval-Augmented Generation) pipelines using Python, Pytest and Ragas.

### ✔ What’s included
- Context Recall & Precision Tests
- RAGAS Score Evaluation
- Latency & Performance metrics
- Prompt consistency checks
- API Test Automation (LangServe / FastAPI)
- PDF → Chunk → Vector DB → LLM flow validation

### ✔ Skills Demonstrated
- Python
- Pytest
- GenAI Testing
- LLM Evaluation Metrics
- RAG Pipeline QA
- Test Design & Reporting
- API + Backend Validation
